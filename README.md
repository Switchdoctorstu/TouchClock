# Arduino TouchClock
TFT Touch Screen clock controller for Orrery

2.4 inch touch shield on Arduino Uno

Current clock displayed
Holds list of dates
Allows editing of dates
Allows date to be selected for display on orrery.
Allows displayed date to be run forward and backward

Display date sent to slave for rendering onto LEDs.

TFT Display: Screen split into 6 rows
row 0:  Buttons 0-5
row 1:  Clock Display
row 2:  Edit up buttons
row 3:  List display
row 4:  Edit down buttons
row 5:  Buttons 50-55
